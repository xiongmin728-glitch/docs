# Regex 正则表达式
字符类

- 任意字符（默认不含换行）

\d 数字 [0-9]

\w 字母数字下划线 [A-Za-z0-9_]

\s 空白（空格、tab 等）

[...] 字符集合：[ABC] 或范围 [A-Z]

[^...] 反集合：[^)] 表示不是右括号的任何字符

- 次数（量词）

* 0 次或多次

+ 1 次或多次

? 0 或 1 次（可选）

{m} 恰好 m 次

{m,} 至少 m 次

{m,n} m 到 n 次

- 位置（锚点）

^ 行/字符串开头

$ 行/字符串结尾

\b 单词边界

- 逻辑

(...) 捕获分组（提取用）

(?:...) 非捕获分组（只分组不提取）

| 或
